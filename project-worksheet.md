# Project Overview

## Project Description

My project will be an app that will list museums around the NYC area and display
event(s) that will take place in that museum. 

## Installation

Dependcies that will be need to get the project running will be:

* express
* morgan
* body-parser
* pg-promise
* ejs
* method-override
